# Ivy's Math World

A 3D math game for Ivy: wander a little world with your character and pet, solve
math problems to earn gems, and spend gems on assets to build your own space.
Family project — Finn briefs, Oscar designs, Nathan builds.

## Getting started
```bash
npm install
npm run dev   # http://localhost:5180
```

## Stack
React 19 · Vite 8 · Tailwind · react-three-fiber / drei / three. iPad-first
(tap-to-move). See `CLAUDE.md` for build state, staging plan, and gotchas.
